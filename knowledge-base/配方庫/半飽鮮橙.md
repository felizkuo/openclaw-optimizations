# 半飽鮮橙配方卡

## 基本資訊
- **名稱**: 半飽鮮橙
- **分類**: 水果茶系列
- **來源**: 抖音
- **提取日期**: 2026-03-11

## 配方材料
| 材料 | 用量 |
|------|------|
| 檸檬 | 2片 |
| 橙子果肉 | 60克 |
| 橙子汁 | 30毫升 |
| 茉莉綠茶 | 120毫升 |
| 直飲水 | 50毫升 |
| 冰糖糖漿 | 40克 |
| 冰塊 | 500克 |
| 橙子（裝飾） | 1片 |

## 製作步驟
1. 在水杯內加入2片檸檬
2. 加入60克橙子果肉
3. 加入冰塊
4. 把橙子錘成吸管能吸上來的狀態
5. 加入橙子汁30毫升
6. 倒入茉莉綠茶120毫升
7. 加入直飲水50毫升
8. 加入冰糖糖漿40克
9. 冰塊加滿到500克線
10. 倒入出品杯
11. 加入1片橙子裝飾
