## 🚀 快速开始

### 步骤 1：环境准备

```bash
# 检查 Python 版本（需要 3.11+）
python --version

# 检查 FFmpeg（需要 5.1.8+）
ffmpeg -version

# 检查 Git
git --version
```

### 步骤 2：克隆工具仓库

```bash
# 克隆小红书 MCP
git clone https://github.com/Algovate/xhs-mcp.git
cd xhs-mcp
pip install -r requirements.txt

# 克隆抖音 MCP Server
git clone https://github.com/yzfly/douyin-mcp-server.git
cd douyin-mcp-server
pip install -r requirements.txt
```

### 步骤 3：配置 API Keys

```bash
# 小红书 Cookie
export XHS_COOKIE=your_cookie_here

# 硅基流动 API
export API_KEY=sk-xxx
```
